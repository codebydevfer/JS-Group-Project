# JS Group Project

## Features
- Username + rooms
- Typing indicator
- Online users list per room
- Message history persisted in `messages.json`
- Simple UI

## Run locally
1. `npm install`
2. `node server.js`
3. Open `http://localhost:3000`

## Deploy
- Start command: `node server.js`
- Make sure the host supplies the `PORT` env var (defaults to 3000)
